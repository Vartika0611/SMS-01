## Student Management System
### Description

The **Student Management System** is a web-based application developed using **Python and Django** to manage student-related data efficiently. The system provides separate roles for **students and administrators**, enabling streamlined academic and administrative operations.

### Features

* Student registration and secure authentication
* Student profile management
* Course and academic record management
* Admin dashboard for managing students and data
* CRUD operations with database integration
* User-friendly and responsive interface

### Technologies Used

* Python
* Django
* HTML, CSS, Bootstrap
* SQLite / MySQL

### Purpose

This project was developed during my **Django Internship at Softpro India** to gain hands-on experience in backend development, database management, and building real-world web applications using the Django framework.
